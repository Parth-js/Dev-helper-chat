# Developer Helper Chat

A modern, responsive chat application designed for developers to communicate, test ideas, or explore a clean messaging interface. Built using **HTML**, **CSS**, and **JavaScript**.  

---

## 🌟 Features

- Responsive and mobile-friendly design.
- Stylish dark theme with clean chat bubbles.
- Differentiates user and bot messages.
- Smooth hover effects and input animations.
- Loader animation to simulate typing or response.
- Easy to customize and extend.
